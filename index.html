A observadora_

Ferramentas utilizadas:

HTML CSS JavaScript

Projeto realizado pela plataforma Alura.
Este projeto de programação consiste em uma personagem animada cujas pupilas acompanham o movimento do cursor do mouse em tempo real.
À medida que o usuário move o mouse pela tela, os olhos da personagem se ajustam dinamicamente, simulando um olhar que segue o ponteiro, criando uma interação visual envolvente e divertida.

Jogo pong_

Ferramentas utilizadas:

HTML CSS JavaScript

Projeto realizado pela plataforma Alura.
Este projeto de programação consiste em recriar o icônico jogo Pong.
Nele, dois jogadores controlam barras verticais que se movem para cima e para baixo, tentando rebater uma bola que se move continuamente pelo espaço de jogo. Cada vez que a bola ultrapassa uma das bordas, o adversário marca um ponto.
O objetivo do jogo é simples e viciante: manter a bola em movimento pelo maior tempo possível e marcar mais pontos que o oponente.
